# 3112-Final-Project
Personal Expense Tracker
Author: James Paek

Project Description:
This is a small Java console-based Personal Expense Tracker. The program allows one user to add expenses, view all expenses, calculate total spending, calculate spending by category, and save/load data using a text file. Built using Java with object-oriented programming principles and file handling.

Files Included:
- Main.java
- ExpenseTracker.java
- Expense.java
- Category.java
- FoodCategory.java
- TransportationCategory.java
- EntertainmentCategory.java
- OtherCategory.java
- FileManager.java
- README.txt

How to Compile:
1. Open a terminal.
2. Navigate to the project folder.
3. Run this command:

javac *.java

How to Run:
After compiling, run this command:

java Main

How to Use:
The program will show a menu with these options:

1. Add Expense
2. View All Expenses
3. View Total Spending
4. View Spending By Category
5. Save Data
6. Save and Exit

Expense data is saved in a file named expenses.txt. The file is created automatically after saving.

Notes:
- This project is intentionally small.
- It supports one user only.
- It does not use a GUI, database, or online features, all local and offline.

## Features
- Add expenses
- View all expenses
- Calculate total spending
- Calculate spending by category
- Save and load expense data from file

## Concepts Used
- Object-Oriented Programming
- Inheritance
